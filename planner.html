    How it works

    Built with plain JavaScript

    This planner stores your tasks as an array of objects, renders them to the page with the DOM API, and saves the array to localStorage after every change — no frameworks or external libraries involved.

    Tips

    • Click the circular checkbox to mark a task complete.
    • Use the filter chips to focus on what's still outstanding.
    • Tasks persist per browser — clearing site data will reset them.
